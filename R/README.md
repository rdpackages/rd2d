# R Files

This directory contains the R implementation and related illustration material.

- `rd2d/` is the R package source root. It contains `DESCRIPTION`, `NAMESPACE`, `R/`, `man/`, and tests.
- `Data/` contains data used by the illustration scripts in this directory.
- `Results/` contains selected generated figures from the illustration scripts.
- `rd2d_illustration.R`, `rd2d_plot_effects.R`, and `rd2d_plot_heatmaps.R` are user-facing examples.

The illustration scripts can be run from either the repository root or this `R/` directory.
